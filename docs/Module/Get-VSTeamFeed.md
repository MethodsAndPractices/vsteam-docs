


# Get-VSTeamFeed

## SYNOPSIS

Returns a list of package feeds for the account.

## SYNTAX

## DESCRIPTION

Get-VSTeamFeed gets all the feeds for the account

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------

```PowerShell
PS C:\> Get-VSTeamFeed
```

This command returns all the package feeds for the account.

## PARAMETERS
