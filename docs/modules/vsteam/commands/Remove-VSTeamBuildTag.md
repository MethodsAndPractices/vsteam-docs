# Remove-VSTeamBuildTag

## SYNOPSIS
Removes the tag from a build.

## SYNTAX

```
Remove-VSTeamBuildTag [-Tags] <String[]> -Id <Int32[]> [-Force] [-ProjectName] <String> [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
Removes the tag from a build.
