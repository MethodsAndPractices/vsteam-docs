# Show-VSTeam

## SYNOPSIS
Opens TFS or AzD site in the default browser.

## SYNTAX

```
Show-VSTeam [<CommonParameters>]
```

## DESCRIPTION
Opens TFS or AzD site in the default browser.

## EXAMPLES

### EXAMPLE 1
```
PS C:\> Show-VSTeam
```

This will open a browser to the TFS or AzD site
