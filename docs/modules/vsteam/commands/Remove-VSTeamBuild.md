# Remove-VSTeamBuild

## SYNOPSIS
Deletes the build.

## SYNTAX

```
Remove-VSTeamBuild -Id <Int32[]> [-Force] [-ProjectName] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Deletes the build.

## EXAMPLES

### EXAMPLE 1
```
PS C:\> Get-VSTeamBuild | Remove-VSTeamBuild -Force
```

This command will delete all builds that are not marked retain indefinitely.
